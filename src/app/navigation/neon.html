This bootstrap navigation directive is using ui-router module. If you want to use another routing system, refer to Routing section.

Usage

navigation: Navigation array.

Creates

Navigation array

[ { name: "Home", link: "app.home" }, { name: "Navigation", subtree: [ { name: "Bootstrap", link: "app.navigation.bootstrap" }, { name: "Neon", link: "app.navigation.neon" } ] } ]

Routing

If you want to switch to another routing system. You can modify all references to
ui-sref-active=""
and
ui-sref=""
in the leaf-neon.tpl.html file to match your routing system.
